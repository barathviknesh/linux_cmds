Linux commands :

1. hostname - machine name
2. ifconfig - ip address
3. data - sys date
4. whoami - current user
5. top - sys info
6. clear - clear the terminal (q)
7. free -m - sys memnory details
8. pwd- present working directory
9. history - display used cmds
10. man - manual man history
11. uname -device os
12. create user - man useradd manual
13. useradd -m bluser (bluser user added)
14. su - switch user
15. su bluser - user switched to bluser
16. cd ~ - go home director 
17. cd - change directory
18. ls - list
19. ls -l - list with details
